Button
======

Graphical user interface of Telegram-bots seems simple, but «The devil is in the details»:

   - How serialize data into `Button`? 
   - How deserialize callback query after catch `Button` click?
   - How determine what `Button` was clicked?
   - How protect `Button` action from fake callback query?
