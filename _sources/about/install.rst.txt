Installation
------------

.. attention::

   Package ``python-telegram-bot`` need to be installed:

    .. code-block:: bash

        pip3 install python-telegram-bot

PyPi contains last stable `Telegram-Bot-Discussion` package:

.. code-block:: bash

   pip3 install telegram-bot-discussion
